## Finance Dashboard Backend

## Tech Stack
1- Node.js
2- Express.js
3- MongoDB (Mongoose)
4- JSON Web Token (JWT)

## User Roles
Viewer
Can only view records and dashboard data
Analyst
Can view, create, and update records
Admin
Has full access including user management

## Installation
Clone the repository

git clone "https://github.com/Anshuman1s/Zorvyn-assignment"

cd backend

## Install dependencies
npm install

## Create a .env file
PORT=5000
MONGO_URI=mongodb://localhost:27017/assignment12
JWT_SECRET=write whatever you want 

## Run Server
npm run dev
